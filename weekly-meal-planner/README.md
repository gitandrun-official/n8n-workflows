## üé¨ About This Workflow

This AI automation does the following:

1. Generates **14 meals for the week** (2 meals per day: lunch and dinner).  
2. Uses the **Google Search tool** to find recipes for each meal.  
3. Extracts and lists:
   - Dish name  
   - Ingredients  
   - Step-by-step recipe  
4. Formats the output for **Monday through Sunday**, including all meals.  
5. Consolidates a **grocery list** with all ingredients and quantities for the week.  
6. Sends the **final weekly plan via email**:
   - Subject matches the weekly header, e.g., `"Monday - Sunday (Nov 3 to Nov 9, 2025)"`  
   - Body is **HTML formatted** with headings, lists, and clear structure for readability
